Getting Started ​

Prerequisites

Here we assume that you are familiar with Lux.jl. If not please take a look at the Lux.jl tutoials.

Boltz.jl is just like Lux.jl but comes with more "batteries included". Let's start by defining an MLP model.

julia
using Lux, Boltz, Random

Multi-Layer Perceptron ​

If we were to do this in Lux.jl we would write the following:

julia
model = Chain(
    Dense(784, 256, relu),
    Dense(256, 10)
)
Chain(
    layer_1 = Dense(784 => 256, relu),  # 200_960 parameters
    layer_2 = Dense(256 => 10),         # 2_570 parameters
)         # Total: 203_530 parameters,
          #        plus 0 states.

But in Boltz.jl we can do this:

julia
model = Layers.MLP(784, (256, 10), relu)
MLP(in_dims=784, hidden_dims=(256,), out_dims=10)(
    block1 = DenseNormActDropoutBlock(
        dense = Dense(784 => 256, relu),  # 200_960 parameters
    ),
    block2 = DenseNormActDropoutBlock(
        dense = Dense(256 => 10),       # 2_570 parameters
    ),
)         # Total: 203_530 parameters,
          #        plus 0 states.

The MLP function is just a convenience wrapper around Lux.Chain that constructs a multi-layer perceptron with the given number of layers and activation function.

How about VGG? ​

Returned Values

The returned value from Vision module functions are a 3 tuple of (model, ps, st). The ps and st are the parameters and states of the model respectively.

Let's take a look at the Vision module. We can construct a VGG model with the following code:

julia
model, _, _ = Vision.VGG(13)
model
Chain(
    layer_1 = Conv((3, 3), 3 => 64, relu, pad=1),  # 1_792 parameters
    layer_2 = Conv((3, 3), 64 => 64, relu, pad=1),  # 36_928 parameters
    layer_3 = MaxPool((2, 2)),
    layer_4 = Conv((3, 3), 64 => 128, relu, pad=1),  # 73_856 parameters
    layer_5 = Conv((3, 3), 128 => 128, relu, pad=1),  # 147_584 parameters
    layer_6 = MaxPool((2, 2)),
    layer_7 = Conv((3, 3), 128 => 256, relu, pad=1),  # 295_168 parameters
    layer_8 = Conv((3, 3), 256 => 256, relu, pad=1),  # 590_080 parameters
    layer_9 = MaxPool((2, 2)),
    layer_10 = Conv((3, 3), 256 => 512, relu, pad=1),  # 1_180_160 parameters
    layer_11 = Conv((3, 3), 512 => 512, relu, pad=1),  # 2_359_808 parameters
    layer_12 = MaxPool((2, 2)),
    layer_13 = Conv((3, 3), 512 => 512, relu, pad=1),  # 2_359_808 parameters
    layer_14 = Conv((3, 3), 512 => 512, relu, pad=1),  # 2_359_808 parameters
    layer_15 = MaxPool((2, 2)),
    layer_16 = FlattenLayer{Nothing}(nothing),
    layer_17 = Dense(25088 => 4096, relu),  # 102_764_544 parameters
    layer_18 = Dropout(0.5),
    layer_19 = Dense(4096 => 4096, relu),  # 16_781_312 parameters
    layer_20 = Dropout(0.5),
    layer_21 = Dense(4096 => 1000),     # 4_097_000 parameters
)         # Total: 133_047_848 parameters,
          #        plus 4 states.

We can also load pretrained ImageNet weights using

julia
model, _, _ = Vision.VGG(13; pretrained=true)
